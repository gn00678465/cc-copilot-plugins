# 繁體中文（台灣慣用）技術詞彙對照

撰寫 commit message 的描述與正文時，應使用下表的台灣慣用譯法，避免使用大陸譯法或英文夾雜。

## 目錄

- [核心詞彙](#核心詞彙)
- [動詞](#動詞)
- [安全／協作](#安全協作)

## 核心詞彙

| 英文 | 繁體中文（台灣慣用） |
|------|---------------------|
| package | 套件 |
| config / configuration | 設定 |
| script | 腳本 |
| dependency | 相依性 |
| component | 元件 |
| module | 模組 |
| function | 函式 |
| variable | 變數 |
| parameter | 參數 |

## 動詞

| 英文 | 繁體中文（台灣慣用） |
|------|---------------------|
| implement | 實作 |
| initialize | 初始化 |
| optimize | 優化 |
| refactor | 重構 |
| validate | 驗證 |
